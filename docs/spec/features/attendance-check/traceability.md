## 사용 규칙

- 새로운 요구사항이 생기면 이 문서에 먼저 ID를 추가한다.
- API 또는 DDL이 변경되면 연결된 행을 함께 갱신한다.
- 빈 칸이 남아 있으면 구현에 들어가지 않는다.
